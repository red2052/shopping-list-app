# 쇼핑 리스트 앱

간단하고 사용하기 쉬운 쇼핑 리스트 웹 애플리케이션입니다.

## 기능

- 아이템 추가/삭제
- 체크박스로 완료 표시
- 로컬 스토리지에 데이터 저장 (브라우저 종료 후에도 유지)
- 총 아이템 수 및 완료 수 통계
- 반응형 디자인

## 사용 방법

1. `shopping-list.html` 파일을 브라우저에서 열기
2. 입력창에 필요한 물건 입력
3. "추가" 버튼 클릭 또는 Enter 키 누르기
4. 체크박스를 클릭하여 완료 표시
5. × 버튼으로 아이템 삭제

## 기술 스택

- HTML5
- CSS3 (Flexbox, Animations)
- Vanilla JavaScript
- LocalStorage API
